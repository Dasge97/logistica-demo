# Logistica y Tarifas

Proyecto nuevo e independiente para validar y operar un motor de decision logistica en tiempo real.

## Documentos principales

- `SPEC.md`: definicion funcional y tecnica de la v1.
- `TASKS.md`: desglose completo de implementacion.

## Objetivo del producto

Construir una aplicacion monolitica en Symfony, desplegable y single-company, capaz de:

- ofrecer solo opciones de entrega viables;
- calcular precio cliente con reglas comerciales;
- calcular coste logistico con reglas operativas;
- seleccionar el vehiculo optimo;
- guardar un snapshot inmutable de cada pedido confirmado.

## Estado actual

El proyecto se encuentra en fase de construccion inicial.

La referencia funcional y tecnica vigente es `SPEC.md`.
